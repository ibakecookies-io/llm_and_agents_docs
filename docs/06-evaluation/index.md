# 06. Оценка качества и evals

Раздел для материалов о том, как проверять качество LLM- и агентных систем до и после внедрения.

## Что хранить в этом разделе

- offline и online evals;
- task success, groundedness, correctness, tool success metrics;
- оценка multi-step и agentic поведения;
- dataset design для evals;
- regression testing для prompt, RAG и agent pipelines.
