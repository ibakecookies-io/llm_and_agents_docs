# 99. Глоссарий

Раздел для коротких определений терминов, которые часто повторяются по всей базе знаний.

## Как использовать глоссарий

- Добавляйте сюда только действительно повторяющиеся термины.
- Держите определения короткими и прикладными.
- Если термин требует длинного объяснения, создайте отдельную заметку и оставьте здесь ссылку.

## Стартовый список терминов

### Agent
Система, в которой модель не только генерирует ответ, но и принимает решения о следующих действиях, может вызывать инструменты и продолжать цикл выполнения до достижения цели.

### Context window
Максимальный объем токенов, который модель может учитывать в одном запросе.

### Embeddings
Векторные представления текста, используемые для поиска похожих фрагментов, кластеризации и retrieval-задач.

### Groundedness
Степень, в которой ответ опирается на предоставленные источники или факты, а не на неподтвержденную генерацию.

### Prompt injection
Попытка изменить поведение модели через вредоносные инструкции во входных данных, документах или внешнем контенте.

### RAG
Подход, при котором модель получает релевантный внешний контекст из базы знаний или поиска перед генерацией ответа.

### Tool calling
Механизм, при котором модель выбирает и вызывает внешнюю функцию, API или инструмент для выполнения действия.
