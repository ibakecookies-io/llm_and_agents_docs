# 02. Prompting и инструкции

Раздел про то, как управлять поведением модели через инструкции, формат входных данных, контекст и структуру взаимодействия.

## Что хранить в этом разделе

- system / developer / user prompts;
- zero-shot, few-shot, chain-of-thought и structured prompting;
- prompt patterns для классификации, генерации, extraction и planning;
- техники снижения ошибок и повышения воспроизводимости;
- проектирование выходных форматов и schema-guided generation.
